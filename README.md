## 📚 Topics Covered

### 1\. Cybersecurity Basics

* CIA Triad: Confidentiality, Integrity, Availability
* Threat Types: Phishing, Malware, DDoS, SQL Injection, Brute Force, Ransomware
* Attack Vectors: Social Engineering, Wireless Attacks, Insider Threats

### 2\. Linux Fundamentals

See [`linux-cheatsheet.md`](./linux-cheatsheet.md) for full command reference.

### 3\. Networking Basics

* OSI Model (7 Layers)
* TCP/IP Protocol Suite
* DNS \& HTTP/HTTPS
* IP Addressing, Subnetting, NAT

### 4\. Cryptography

* Symmetric vs Asymmetric Encryption
* Hashing: MD5, SHA256
* Digital Certificates \& SSL/TLS
* Hands-on: OpenSSL encrypt/decrypt

### 5\. Tools Explored

|Tool|Use Case|
|-|-|
|Wireshark|Packet capture \& analysis|
|Nmap|Network scanning|
|Burp Suite|Web proxy \& HTTP interception|
|Netcat|Network debugging \& port listening|
